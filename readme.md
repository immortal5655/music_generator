# 运用LSTM生成一段旋律

### 1. 运行方法

- 安装 music21,并配置环境变量

- 运行 preprocess.py
- 运行 train.py
- 运行 melody_generator.py (如果你想根据一小段旋律来根据网络生成后面的旋律，只需更改"seed",数字表示单音，"_"表示持续时间，以16分音符作为持续单位)
